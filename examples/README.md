# Tree DP Examples

This folder contains example DP formulas for the Tree DP Editor.

## Examples

1. **subtree_size.txt** - Calculate the size of each subtree
2. **subtree_sum.txt** - Calculate the sum of values in each subtree
3. **max_independent_set.txt** - Maximum Independent Set problem
4. **min_vertex_cover.txt** - Minimum Vertex Cover problem
5. **tree_diameter.txt** - Find the diameter of the tree
6. **sum_of_distances.txt** - Sum of distances using rerooting DP
7. **tree_matching.txt** - Maximum matching in a tree
8. **tree_coloring.txt** - Tree coloring with k=3 colors
9. **longest_edge_path.txt** - Longest path with edge weights

## Usage

These examples can be loaded directly in the Tree DP Editor by selecting them from the "Examples..." dropdown menu.

You can also use these as templates to create your own DP formulas.
